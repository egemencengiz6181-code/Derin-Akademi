# Derin Akademi — Kurumsal Web Sitesi

Derin Akademi, insan kaynakları alanında uzmanlaşmış danışmanlık ve kurumsal eğitim şirketinin resmi web sitesidir.

## 🚀 Proje Yapısı

```
├── index.html              # Ana sayfa
├── hakkimizda.html         # Hakkımızda
├── kurslar.html            # Eğitimler listesi
├── danismanlik.html        # Danışmanlık listesi
├── blog.html               # Blog listesi
├── kariyer.html            # Kariyer
├── iletisim.html           # İletişim
├── blog/                   # Blog yazıları
├── egitimler/              # Eğitim detay sayfaları
├── danismanliklar/         # Danışmanlık detay sayfaları
├── css/
│   └── main.css            # Ortak stil dosyası
├── js/
│   ├── main.js             # Ortak JavaScript
│   └── i18n.js             # Dil desteği (TR/EN)
└── public/
    └── logos/              # Müşteri logoları
```

## 🛠 Teknolojiler

- **HTML5** — Semantik yapı
- **CSS3** — Custom Properties, Grid, Flexbox, Animasyonlar
- **Vanilla JS** — DOM manipülasyonu, Intersection Observer
- **Google Fonts** — Inter & Montserrat

## 📋 Özellikler

- Tamamen responsive tasarım (mobil öncelikli)
- Dark mode tasarım (navy + teal renk paleti)
- Scroll reveal animasyonları
- Sticky navigasyon
- TR/EN dil desteği
- Blog ve içerik yönetimi
- Danışmanlık ve eğitim detay sayfaları
- SEO optimizasyonu (Open Graph, Twitter Card, Canonical)
